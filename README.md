Installation process on Ubuntu (sorry don't know how installation works on mac or windows)

sudo apt update
sudo apt install nodejs

Check node install:
node -v

sudo apt install sqlite3

Check sqlite3 install:
sqlite3 --version

sudo apt install npm
npm init -y
npm install sqlite3
